## Goal

Stand up the local dashboard process: a Node/Express service that runs as a PM2 daemon (`hive-dashboard`), binds `0.0.0.0:7777`, and gates every request behind a bearer-token auth middleware that reads `HIVE_DASHBOARD_TOKEN` from the framework's root `.env` (which C.7's migration handler populates).

D.1 ships the **skeleton only**: app boot, dotenv loading, auth middleware, one `/api/health` endpoint, PM2 ecosystem entry. No agent endpoints, no doctor endpoint, no SSE — those are D.2/D.3. The skeleton is what every later D leaf builds on, so the contracts here matter:

1. **Dashboard process is its own pnpm project** under `dashboard/` (sibling to `services/hive-releases-api/`). Independent `package.json`, `pnpm-lock.yaml`, `node_modules/`. Not part of the root TypeScript build (the framework runner). The dashboard is plain runtime JS — `node dashboard/index.js`, no build step.
2. **Framework state is read via `process.cwd()` relative paths.** PM2 launches the dashboard with `cwd: __dirname` (which resolves to the framework root because `ecosystem.config.cjs` lives there). Dashboard endpoints in D.2+ resolve `path.join(process.cwd(), 'agents', ...)`, `path.join(process.cwd(), 'data', 'runner-events.log')`, etc.
3. **Root `.env` is the single source of truth** for the dashboard's runtime config. `dotenv.config({ path: path.resolve(__dirname, '..', '.env') })` at top of `dashboard/index.js` loads `~/neato-hive/.env`.
4. **Auth middleware is the GLOBAL default** — every Express route runs through it. `/api/health` is the **only documented unauthenticated exception** (used by Tailscale/PM2 healthchecks). Future endpoints inherit auth without opt-in.
5. **No new shell-tool deps.** Express + dotenv are the only production dependencies. Tests use native `node --test` + native `fetch` (Node 22+ has both — no devDeps).

## Architectural givens (carried)

- **Binding:** `0.0.0.0:7777`. Port configurable via `HIVE_DASHBOARD_PORT` env var (defaults 7777). The literal `0.0.0.0` is locked (Tailscale exposes the local-network bind to the user's mesh; `127.0.0.1` would be unreachable).
- **Token format:** 64 hex chars (256-bit), generated by C.7's `openssl rand -hex 32`. Auth middleware compares using **`crypto.timingSafeEqual`** — never `===` (timing-attack mitigation). Length mismatch is a fast-fail.
- **Token absence is a hard error.** If `HIVE_DASHBOARD_TOKEN` is unset OR empty when the dashboard process boots, log a fatal error and exit non-zero. PM2 will autorestart per ecosystem config; user fix is to run `hive update` or set the env var manually. Failing-loud beats failing-silent (where an empty token would be `''` and any client sending `Bearer ` matches).
- **Auth header parsing:** Bearer-only. `Authorization: Bearer <token>`. Case-insensitive on `Bearer` (express-style). Tokens with leading/trailing whitespace are rejected (no `trim()` — preserves the security invariant that the user's token is byte-exact what's in `.env`).
- **Health endpoint shape:** `GET /api/health` → `200 { status: "ok", uptime_s: <int>, version: <string>, ts: <iso8601> }`. Version comes from `package.json` (the framework's, not the dashboard's). `uptime_s` from `process.uptime()`. `ts` from `new Date().toISOString()`.
- **No request logging in D.1** — D.2 will introduce structured logging if needed. D.1 keeps the logs minimal (just the boot message and any uncaught errors).
- **Graceful shutdown:** SIGTERM closes the HTTP server. PM2 restart cycles are clean.
- **PM2 entry locked:** `name: "hive-dashboard"`, `script: "dashboard/index.js"`, `cwd: __dirname`, `instances: 1`, `exec_mode: "fork"`, `autorestart: true`, `max_restarts: 10`, `min_uptime: "10s"`, `max_memory_restart: "300M"`, `watch: false`, `env: { NODE_ENV: "production" }`. Mirrors `hive-runner` shape for consistency.
- **Worker MUST NOT exec PM2 verbs** (carried from C.7). Worker confirms by file inspection only that the new ecosystem entry is well-formed; live PM2 reload is **owner ceremony post-merge**.
- **No live-bind in worker tests** — tests boot the Express app via `app.listen(0, ...)` to grab an ephemeral port from the OS, never `7777`. Avoids port collisions on the worker's machine.

### A.3 — `dashboard/index.js` (entry point)

```javascript
'use strict';

const path = require('path');

// Load the framework's root .env (where C.7's migration handler writes
// HIVE_DASHBOARD_TOKEN). PM2 launches us with cwd: <framework-root>, but
// belt-and-suspenders we use __dirname to resolve regardless of cwd.
require('dotenv').config({ path: path.resolve(__dirname, '..', '.env') });

const { createApp } = require('./app');

const PORT = parseInt(process.env.HIVE_DASHBOARD_PORT || '7777', 10);
const HOST = '0.0.0.0';

// Token presence is a hard precondition. Empty/unset token would mean any
// `Authorization: Bearer ` (literal empty) matches — fail loud instead.
if (!process.env.HIVE_DASHBOARD_TOKEN || process.env.HIVE_DASHBOARD_TOKEN.length < 32) {
  console.error('[hive-dashboard] FATAL: HIVE_DASHBOARD_TOKEN is unset or too short.');
  console.error('[hive-dashboard] Run `hive update` (which triggers v1.5.0 migration) or set the var manually.');
  process.exit(1);
}

const app = createApp({ token: process.env.HIVE_DASHBOARD_TOKEN });

const server = app.listen(PORT, HOST, () => {
  console.log(`[hive-dashboard] listening on ${HOST}:${PORT}`);
});

// Graceful shutdown for PM2 reloads
function shutdown(signal) {
  console.log(`[hive-dashboard] received ${signal}, closing server...`);
  server.close((err) => {
    if (err) {
      console.error('[hive-dashboard] error during close:', err);
      process.exit(1);
    }
    process.exit(0);
  });
  // Hard-exit fallback if close hangs
  setTimeout(() => {
    console.error('[hive-dashboard] close timeout — forcing exit');
    process.exit(1);
  }, 10000).unref();
}

process.on('SIGTERM', () => shutdown('SIGTERM'));
process.on('SIGINT', () => shutdown('SIGINT'));
```

### A.4 — `dashboard/app.js` (app factory)

Separating `createApp(opts)` from the listener lets tests construct the app with an ephemeral-port `app.listen(0)` and import the same routes the production process uses.

```javascript
'use strict';

const express = require('express');
const { createAuthMiddleware } = require('./middleware/auth');
const healthRouter = require('./routes/health');

/**
 * createApp({ token }) — constructs the Express app.
 *
 * Globals applied:
 *   - JSON body parsing (1MB cap)
 *   - Bearer-token auth middleware (everywhere except /api/health)
 *
 * @param {Object} opts
 * @param {string} opts.token — the bearer token (raw 64-hex from .env)
 * @returns {express.Express}
 */
function createApp({ token }) {
  if (!token) {
    throw new Error('createApp: token is required');
  }

  const app = express();

  app.use(express.json({ limit: '1mb' }));

  // Public health probe — bypasses auth so PM2 / Tailscale healthchecks work.
  app.use('/api/health', healthRouter);

  // Auth middleware applies to every route below this line.
  const auth = createAuthMiddleware(token);
  app.use(auth);

  // No D.1 routes beyond /api/health; D.2+ mounts here.

  // 404 fallback (after middleware so unauth'd requests for unknown paths
  // get 401 not 404 — avoids leaking which paths exist)
  app.use((req, res) => {
    res.status(404).json({ error: 'not_found', path: req.path });
  });

  // JSON error handler (so thrown errors don't leak stack traces)
  // eslint-disable-next-line no-unused-vars
  app.use((err, req, res, next) => {
    console.error('[hive-dashboard] error:', err);
    res.status(500).json({ error: 'internal_error' });
  });

  return app;
}

module.exports = { createApp };
```

### A.5 — `dashboard/middleware/auth.js` (token middleware)

```javascript
'use strict';

const crypto = require('crypto');

/**
 * createAuthMiddleware(expectedToken)
 *
 * Returns a connect-style middleware that verifies the incoming
 * `Authorization: Bearer <token>` header matches `expectedToken`
 * via `crypto.timingSafeEqual`.
 *
 * Reject (401) if:
 *   - Header missing
 *   - Header doesn't start with `Bearer ` (case-insensitive on Bearer)
 *   - Token length doesn't match expectedToken (fast-fail before
 *     timingSafeEqual which requires equal-length buffers)
 *   - Token bytes don't match
 *
 * Never logs the token or its length.
 */
function createAuthMiddleware(expectedToken) {
  if (!expectedToken || typeof expectedToken !== 'string') {
    throw new Error('createAuthMiddleware: expectedToken must be a non-empty string');
  }

  const expectedBuf = Buffer.from(expectedToken, 'utf8');

  return function authMiddleware(req, res, next) {
    const header = req.headers['authorization'] || req.headers['Authorization'];
    if (!header || typeof header !== 'string') {
      return res.status(401).json({ error: 'unauthorized' });
    }

    // Case-insensitive Bearer prefix; preserve token bytes verbatim
    const match = header.match(/^Bearer\s+(.+)$/i);
    if (!match) {
      return res.status(401).json({ error: 'unauthorized' });
    }

    const presentedToken = match[1];
    const presentedBuf = Buffer.from(presentedToken, 'utf8');

    // Length mismatch fast-fail — timingSafeEqual throws on unequal lengths
    if (presentedBuf.length !== expectedBuf.length) {
      return res.status(401).json({ error: 'unauthorized' });
    }

    if (!crypto.timingSafeEqual(presentedBuf, expectedBuf)) {
      return res.status(401).json({ error: 'unauthorized' });
    }

    return next();
  };
}

module.exports = { createAuthMiddleware };
```

### A.6 — `dashboard/routes/health.js` (health endpoint)

```javascript
'use strict';

const express = require('express');
const path = require('path');
const fs = require('fs');

const router = express.Router();

let cachedFrameworkVersion = null;
function getFrameworkVersion() {
  if (cachedFrameworkVersion) return cachedFrameworkVersion;
  try {
    const pkgPath = path.resolve(process.cwd(), 'package.json');
    const pkg = JSON.parse(fs.readFileSync(pkgPath, 'utf8'));
    cachedFrameworkVersion = pkg.version || 'unknown';
  } catch (err) {
    cachedFrameworkVersion = 'unknown';
  }
  return cachedFrameworkVersion;
}

router.get('/', (req, res) => {
  res.status(200).json({
    status: 'ok',
    uptime_s: Math.floor(process.uptime()),
    version: getFrameworkVersion(),
    ts: new Date().toISOString(),
  });
});

module.exports = router;
```

**Why read the framework version, not the dashboard package version:** the user-meaningful version IS the framework version (`hive --version` reports it; `package.json` at framework root is the source). Dashboard's own package.json mirrors `1.5.0` as a coincidence of release alignment but the framework root is canonical.

### A.8 — `dashboard/test/auth.test.js`

```javascript
'use strict';

const test = require('node:test');
const assert = require('node:assert');
const { createAuthMiddleware } = require('../middleware/auth');

const VALID_TOKEN = 'b'.repeat(64);

function makeReqRes(headers = {}) {
  const req = { headers };
  let statusCode = null;
  let body = null;
  const res = {
    status(code) { statusCode = code; return this; },
    json(payload) { body = payload; return this; },
  };
  let nextCalled = false;
  const next = () => { nextCalled = true; };
  return { req, res, next, get: () => ({ statusCode, body, nextCalled }) };
}

test('auth: missing header → 401', () => {
  const mw = createAuthMiddleware(VALID_TOKEN);
  const ctx = makeReqRes({});
  mw(ctx.req, ctx.res, ctx.next);
  const r = ctx.get();
  assert.strictEqual(r.statusCode, 401);
  assert.strictEqual(r.nextCalled, false);
});

test('auth: malformed header (no Bearer) → 401', () => {
  const mw = createAuthMiddleware(VALID_TOKEN);
  const ctx = makeReqRes({ authorization: VALID_TOKEN });  // missing Bearer
  mw(ctx.req, ctx.res, ctx.next);
  assert.strictEqual(ctx.get().statusCode, 401);
});

test('auth: wrong-length token → 401 (timingSafeEqual fast-fail)', () => {
  const mw = createAuthMiddleware(VALID_TOKEN);
  const ctx = makeReqRes({ authorization: 'Bearer short' });
  mw(ctx.req, ctx.res, ctx.next);
  assert.strictEqual(ctx.get().statusCode, 401);
});

test('auth: same-length wrong token → 401', () => {
  const mw = createAuthMiddleware(VALID_TOKEN);
  const ctx = makeReqRes({ authorization: `Bearer ${'c'.repeat(64)}` });
  mw(ctx.req, ctx.res, ctx.next);
  assert.strictEqual(ctx.get().statusCode, 401);
});

test('auth: correct token → next() called', () => {
  const mw = createAuthMiddleware(VALID_TOKEN);
  const ctx = makeReqRes({ authorization: `Bearer ${VALID_TOKEN}` });
  mw(ctx.req, ctx.res, ctx.next);
  const r = ctx.get();
  assert.strictEqual(r.nextCalled, true);
  assert.strictEqual(r.statusCode, null);
});

test('auth: case-insensitive Bearer prefix → next() called', () => {
  const mw = createAuthMiddleware(VALID_TOKEN);
  const ctx = makeReqRes({ authorization: `bearer ${VALID_TOKEN}` });
  mw(ctx.req, ctx.res, ctx.next);
  assert.strictEqual(ctx.get().nextCalled, true);
});

test('auth: leading whitespace in token → 401 (no trim)', () => {
  const mw = createAuthMiddleware(VALID_TOKEN);
  const ctx = makeReqRes({ authorization: `Bearer  ${VALID_TOKEN}` });  // double space
  mw(ctx.req, ctx.res, ctx.next);
  // Bearer regex captures "(.+)" — extra space ends up in token, length mismatch
  assert.strictEqual(ctx.get().statusCode, 401);
});

test('createAuthMiddleware: empty token throws', () => {
  assert.throws(() => createAuthMiddleware(''), /non-empty string/);
  assert.throws(() => createAuthMiddleware(null), /non-empty string/);
  assert.throws(() => createAuthMiddleware(undefined), /non-empty string/);
});
```

## E. Standing rules

- **DON'T HALF-SHIP** — full skeleton (app + middleware + health route + tests + ecosystem entry) in single PR. No "we'll add the auth tests in D.2."
- **DO NOT MERGE** — house-md
- **DO NOT EXEC PM2** — worker code MUST NOT invoke `pm2 startOrReload`, `pm2 restart`, `pm2 reload`, `pm2 delete`, `pm2 save`, `pm2 stop`, or `pm2 kill`. PM2 ecosystem reload is owner ceremony post-merge.
- **DO NOT EXTEND DEPENDENCIES** — production deps locked at `express` + `dotenv`. Adding any other dep requires explicit spec amendment. Tests use native `node --test` + native `fetch` only.
- **DO NOT BIND TO 7777 IN TESTS** — tests use `app.listen(0, ...)` for ephemeral port allocation OR explicit `HIVE_DASHBOARD_PORT=177xx` for boot smoke tests. Real production bind is `0.0.0.0:7777`.
- **DO NOT LOG OR ECHO TOKENS** — no `console.log(token)`, no error messages that include the token bytes, no diagnostic dumps that print env vars.
- **`crypto.timingSafeEqual` IS THE COMPARISON** — never `===` for token compare, never `Buffer.compare`. Length mismatch is a fast-fail before timingSafeEqual.
- **HALT-and-ping rule** — pre-flight surprises (dashboard/ exists, ecosystem already has hive-dashboard, Node < 22, pnpm absent) stop the worker.
- **`gh repo clone` not SSH** for any fresh clones; remote-URL check before cleanup.
- **on-complete prompt is bob-aimed** — pings house-md `kind=delegation` when DONE block emitted.
- **dirty git status whitelist** — agents/, runtime, data/, docs/TASK.md, root pnpm-lock.yaml drift OK pre-flight (but `dashboard/pnpm-lock.yaml` MUST be committed clean).
- **No new shell-tool deps** — Node + pnpm + curl + jq all standard. Wizard-dep-rule clean.
